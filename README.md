# Movie Listing App

A modern Vue 3 application for browsing and managing your favorite movies. Built with TypeScript and Composition API.

## Features

- Browse movies with real-time search
- Save favorite movies with persistent storage
- Responsive design for all devices
- Advanced search functionality
- Pagination support
- Modern and clean UI

## Tech Stack

- Vue 3 with Composition API
- TypeScript
- Vuex 4 for state management
- SCSS for styling
- Vite for build tooling
- Vitest for unit testing
- Cypress for integration testing
- Axios for API calls

## Getting Started

### Prerequisites

- Node.js (v20 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/ngodichat/movie-listing.git

2. Install dependencies
cd movie-listing
npm install

3. Start development server
npm run dev

4. Build for production
npm run build

5. Run unit test
npm run test

6. Run integration test
npm run test:e2e
